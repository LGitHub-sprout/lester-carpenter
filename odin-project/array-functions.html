JavaScript Built‐in Functions

An array of performing artists and a few built‐in JavaScript methods for manipulating it.

Array Methods

toString()
Returns a string representing the specified array and its elements.
pop()
Removes the last element from an array and returns the popped element(s) which changes the length of the array (mutates).
push()
Adds one or more elements to the end of an array and returns the new length of the array.
slice(start, end)
Returns a portion of an array into a new array object selected from start to end (not inclusive) where start and end represent the index of items in that array. (Immutable).
splice(start, deleteCount, item1, item2, itemN)
Changes the contents of an array by removing or replacing existing elements and/or adding new elements in place (mutable). To access part of an array without modifying it, see slice().
The return value is an array containing the deleted elements. If no elements are removed, an empty array is returned.
