JavaScript Callback Functions

Standard Built-in Objects: Array

In this exercise, I'll build a callback function similar to a JavaScript (JS) built-in method, based on Briggs Elsperger's post, Understanding Callbacks called map() which, according to MDN's reference page, "creates a new array populated with the results of calling a provided function on every element in the calling array".

Although I built a working version once before, this time I'll be challenging myself to rebuild it from memory as well as using problem solving techniques as a means to better understand the underlying principles of it as well as JS in general.

I'm starting with a few basic pieces of info from notes, so I know I need to create a myMap() function that returns a new array that pushes the results of a loop that then applies a function to each element in an array. Just writing it out like that is a bit confusing to me, but I have a better idea of what I need than how well I'm able to describe it, unfortunately. Hopefully, I can come back and rewrite it better, but until then I'll move forward building the first pieces step by step.

The myMap(arr, callback) function takes two parameters, an array and a callback function.

The callbackResult is what the callback returns.

The above array of names has been manipulated, based on Briggs Elsperger's post, Understanding Callbacks, which sets each array item to all uppercase using a custom function that loops through any provided array applying the JavaScript .toUpperCase() string method to each item of the array.

filter()

As the MDN web docs state, the filter() method creates a new array with all elements that pass the test implemented by the provided function.

In this case, I'm filtering the array with the startsWith() method inside a callback function which gets passed to filter() in what's called an arrow function expression.

You might have noticed that all the names begin with the letter 'L' which is what is passed as the criteria to test for; therefore, these are all the elements in the array that begin with that letter.
